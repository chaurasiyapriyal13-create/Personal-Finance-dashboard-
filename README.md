# Personal Finance Power BI Dashboard

This project is an interactive Power BI dashboard designed to analyze personal income, expenses, and savings behavior.

## Dashboard Features
- KPI-style layout
- Category-wise expense analysis
- Monthly income vs expense trends
- Interactive slicers and filters
- Clean, professional UI design

## Tools Used
- Power BI
- DAX
- CSV Dataset
- Data Visualization

## Files Included
- Power BI (.pbix) file
- Dataset (CSV)
- Dashboard screenshots

## Purpose
This project was built as a learning exercise to practice dashboard design, interactivity, and storytelling using Power BI.
